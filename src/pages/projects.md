# Projects

Below is a showcase of open-source software I've built.

## Python & Data Tools

From spectral similarity algorithms to financial modeling, these projects highlight my focus on creating robust scientific and analytical software.

### [MassFlow](https://github.com/janusson/massflow)

A Python package designed for mass spectrometry data analysis featuring custom spectral similarity scoring and robust data visualization tools.

### [PySharpe](https://github.com/janusson/pysharpe)

A quantitative financial tool built to optimize portfolio allocations. Designed for dashboarding, it calculates Sharpe ratios for specific ETFs while accounting for real-world variables like tax drag and FX fee assessments.

## Lab Automation & Method Development

- Development of a bespoke potentiometric chloride ion-selective electrode (ISE) assay to eliminate halogen contamination in industrial analytical workflows.
- Management of a $515,000 CAD capital procurement for a triple-quadrupole ICP-MS and ion chromatography (IC) facility, overseeing vendor selection, installation, and commissioning.
